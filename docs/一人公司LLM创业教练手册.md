# 一人公司 LLM 创业教练手册（从小白到可卖课老师）

> 目标读者：零基础但有强烈创业意愿，想成为“能做产品 + 能带学员 + 能成交课程 + 能持续迭代”的一人公司 CEO。

---

## 0. 总体战略：你要成为“教育型 AI 创业者”

你的终局不是“会写几个提示词”，而是形成 **可复制、可交付、可迭代** 的商业系统：

- **商业层**：稳定获客、转化、复购、转介绍。
- **产品层**：课程 + 训练营 + 社群 + 工具/模板的组合产品矩阵。
- **能力层**：你自己从 0 到 1 训练/部署 MiniMind 风格 LLM 的硬实力。
- **品牌层**：你在细分领域里被认知为“会落地、讲得清、交付稳”的老师。

一句话战略：

**先卖“明确结果”，再卖“进阶能力”，最后卖“系统化陪跑”。**

---

## 1. 创业定位：先定赛道，不要先卷流量

### 1.1 你的价值主张（Value Proposition）

推荐你采用以下定位模板：

> 我帮助【某类人群】在【某个时限】内，从【当前状态】到【目标结果】，通过【你的方法】实现【可见成果】。

示例：

> 我帮助零基础技术从业者在 8 周内完成“可部署可上线可迭代”的个人 LLM 项目，并具备课程化输出能力。

### 1.2 细分人群选择（先选一个）

可优先选择以下单一人群切入：

1. 希望转 AI 岗的开发者。
2. 想做知识付费的技术博主。
3. 传统行业数字化负责人（需要私有模型能力）。

规则：**一次只打一个人群，一次只承诺一个核心结果。**

### 1.3 你的“反同质化”标签

你必须拥有至少一个强标签：

- “真的会从 0 训练 LLM，而非只会调用 API”
- “课程里带你完整部署上线，不止停留在 Notebook”
- “每个学员都有可展示项目与答辩”

## 2. 商业模式设计：一人公司也要有现金流梯度

### 2.1 产品梯度（由低到高）

1. **引流产品（9.9~99）**：公开课、资料包、7 天挑战营。
2. **标准产品（999~3999）**：系统课 + 作业 + 社群答疑。
3. **高客单（5999~19999）**：训练营 + 项目陪跑 + 求职/业务落地辅导。
4. **B2B 服务（2w+）**：企业内训、私有化部署咨询。

### 2.2 收入结构建议

目标上，单月收入来源建议：

- 50% 来自标准课（稳定）
- 30% 来自训练营/高客单（利润）
- 20% 来自咨询或企业服务（现金流补充）

### 2.3 最小可行财务模型（MVP）

你可先以月目标 5 万设计：

- 标准课 1999 元，成交 15 人 ≈ 29985
- 高客单 7999 元，成交 2 人 ≈ 15998
- 资料包/低价课 ≈ 5000+

合计约 5 万。达标后再扩量，而非一开始追百万。

## 3. 卖课系统：从流量到成交的闭环

### 3.1 流量内容三件套

每天内容不求多，求结构统一：

1. **认知类**：讲清“为什么要学本能力”。
2. **案例类**：展示“学员/你自己做成了什么”。
3. **转化类**：明确“谁适合报名、报名后得到什么”。

### 3.2 内容脚本模板（短视频/图文通用）

- 开头 5 秒：戳痛点（“只会调 API，找不到差异化”）
- 中段 30 秒：给方法（“3 步做出可上线 LLM demo”）
- 结尾 10 秒：给行动（“评论区领项目清单/进训练营”）

### 3.3 成交流程 SOP

1. 公域内容引流到私域（企微/社群/邮件）。
2. 私域做 3 次价值触达（案例、答疑、路径图）。
3. 小班直播成交（讲问题 -> 讲方案 -> 讲证据 -> 讲名额）。
4. 48 小时限时转化（截止机制 + 风险逆转）。

### 3.4 风险逆转（降低犹豫）

- 7 天不满意可退（有边界规则）
- 提供里程碑保障（未完成阶段目标可延学）
- 明确“可交付成果列表”

## 4. CEO 视角：你一个人也要有组织系统

### 4.1 一人公司的五大岗位

你要扮演以下角色（按时间切换）：

- CEO（定方向、定优先级）
- CPO（课程/产品设计）
- CMO（营销与转化）
- COO（交付与学员运营）
- CTO（技术与平台）

### 4.2 周节奏（建议）

- 周一：商业复盘 + 指标更新
- 周二：课程内容生产
- 周三：公开内容发布 + 直播准备
- 周四：直播成交 + 私域跟进
- 周五：学员交付 + 答疑
- 周六：技术研发（MiniMind 项目推进）
- 周日：休整 + 下周规划

### 4.3 CEO 每周必看指标

- 流量：新增线索数、线索成本
- 转化：咨询率、成交率、退款率
- 交付：完课率、作业提交率、NPS
- 财务：收入、毛利、现金流可支撑月数

## 5. 学习系统：从小白到“老师级”能力地图

### 5.1 三层能力模型

1. **理解层**：看懂 LLM 基础原理与训练流程。
2. **实操层**：能训练、微调、部署、评估一个模型。
3. **教学层**：能把复杂概念结构化讲清并形成课程。

### 5.2 学习方法（反低效）

- 每学一个概念必须有“代码验证”。
- 每完成一个阶段必须有“可展示作品”。
- 每周输出 1 次“教学型内容”（文章/视频/直播）。

### 5.3 12 个月成长路线（建议）

- 1~3 个月：打基础（Python、PyTorch、Transformer）
- 4~6 个月：MiniMind 全流程复现（预训练/SFT/RL）
- 7~9 个月：部署 + 产品化 + 学员试点交付
- 10~12 个月：课程体系成型 + 稳定成交 + 品牌强化

## 6. 本项目（MiniMind）作为核心竞争力：重点执行方案

你课程的“护城河”来自：**你不仅讲概念，而是带学员完成完整项目闭环**。

### 6.1 项目价值定位（对外表达）

“带学员从 0 开始打造个人 LLM：可训练、可部署、可上线、可迭代。”

这句话要反复出现在你的页面、直播、私域介绍里。

### 6.2 教学主线（建议 8 大模块）

1. 环境与工程准备（依赖、GPU、目录结构）
2. 数据集与 tokenizer 训练
3. 预训练（Pretrain）
4. 有监督微调（SFT/LoRA）
5. 偏好优化（DPO/PPO/GRPO/SPO）
6. 推理与 API 服务化
7. Web Demo 与上线发布
8. 评估、监控与迭代

### 6.3 学员交付件（必须可见）

每位学员至少交付：

- 训练日志与 loss 曲线
- 一版可调用 API 服务
- 一个可演示的 Web 界面
- 一份迭代计划（下个版本要优化什么）

### 6.4 课程差异化设计

你要讲出“为什么这么做”：

- 为什么先让学员跑通全流程，再做细节优化。
- 为什么从最小模型开始，降低算力门槛。
- 为什么把部署和迭代纳入课程，而不止训练。

### 6.5 你的标准教学话术（可复用）

- “先可用，再优化。”
- “先闭环，再扩展。”
- “先项目结果，再理论深挖。”

这三句话是你课程文化的核心。

## 7. 课程产品设计（可直接落地）

### 7.1 课程结构模板

- 模块 1：认知破冰（为什么这门课值得学）
- 模块 2：基础搭建（环境与原理）
- 模块 3：核心实操（MiniMind 主干）
- 模块 4：项目实战（从训练到上线）
- 模块 5：成果发布（演示、答辩、复盘）

### 7.2 训练营交付节奏（8 周示例）

- Week1：环境+代码跑通
- Week2：数据与 tokenizer
- Week3：预训练入门
- Week4：SFT/LoRA
- Week5：偏好优化
- Week6：部署 API
- Week7：前端演示+上线
- Week8：项目答辩+简历/作品包装

### 7.3 教学材料清单

- 学习地图（路线图）
- 每周作业模板
- 常见报错手册
- 直播答疑题库
- 演示项目模板

## 8. 学员运营：口碑和复购的发动机

### 8.1 社群运营核心动作

- 日打卡（保持学习节奏）
- 周复盘（强化成长感）
- 月展示（公开成果增强认同）

### 8.2 学员分层运营

- A 类：执行力高 -> 给挑战任务
- B 类：中等 -> 给标准任务
- C 类：易掉队 -> 给陪伴提醒 + 小步任务

### 8.3 续费与转介绍机制

- 结营前两周启动续费咨询
- 优秀学员案例视频化
- 设计“推荐奖励 + 作品曝光”双激励

---

## 9. 营销系统：你要做“可复利”的内容资产

### 9.1 选平台原则

先选 1 个主平台 + 1 个备份平台，不要全平台开战。

### 9.2 内容资产分层

- 顶层：世界观内容（理念、趋势判断）
- 中层：方法论内容（步骤、框架）
- 底层：案例内容（项目过程、结果对比）

### 9.3 你的品牌关键词（建议）

- 从 0 到 1 LLM 实战
- 可部署可上线可迭代
- 一人公司 AI 创业

---

## 10. 风险管理：一人公司必须防守

### 10.1 法务与合规基础

- 明确课程服务协议与退款规则
- 资料版权声明（禁止未经授权传播）
- 学员隐私与数据保护规则

### 10.2 经营风险

- 不把全部收入押在单次直播
- 不把全部流量押在单个平台
- 保留 6 个月基础运营现金流

### 10.3 创始人风险

- 防过劳：固定休息日
- 防迷失：每月战略回顾
- 防焦虑：只和“上月自己”比较

---

## 11. 90 天执行计划（你可以今天就开始）

### 第 1 阶段（1~30 天）：打地基

- 完成个人定位与产品定义
- 输出 10 条内容（认知/案例/转化各 3+）
- MiniMind 跑通最小闭环（训练到推理）
- 设计首个低价引流产品

### 第 2 阶段（31~60 天）：做成交

- 每周 1 次直播分享
- 建立私域跟进 SOP
- 完成首批种子学员交付
- 产出 3 个学员案例素材

### 第 3 阶段（61~90 天）：做复制

- 课程标准化（教案、作业、答疑库）
- 搭建转介绍机制
- 完成一次完整训练营交付
- 复盘并迭代下一期版本

---

## 12. 你的每日执行清单（极简版）

每天做 5 件事：

1. 学习 1 小时（技术）
2. 产出 1 条内容（营销）
3. 跟进 5 个线索（销售）
4. 服务 3 个学员（交付）
5. 复盘 15 分钟（经营）

长期坚持这 5 件事，能力、现金流、品牌会同步增长。

---

## 13. 给你的创业教练寄语

你现在最需要的不是“完美计划”，而是“可持续执行系统”。

请记住三句话：

1. **先完成，再完美。**
2. **先拿结果，再讲梦想。**
3. **先服务好第一批学员，再扩大规模。**

当你真的能带着学员从 0 做出可运行 LLM 项目，你就不是“学 AI 的人”，而是“能创造价值的 AI 创业者”。
